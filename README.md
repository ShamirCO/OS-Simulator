# OS-Simulator

Please make sure you have installed the following NuGet packages:
  -CoreCLR-NCalc by Sebastian Klose.
  -Newtonsoft.Json by James Newton-King.
  
 The system reacts to certain keys, such as:
  -I. Interruption.
  -E. Error.
  -P. Pause.
  -C. Continue.
  -N. New.
  -T. Process table.
  -M. Page table.
  -S. Suspended.
  -R. Return.
  -U. Virtual memory.
